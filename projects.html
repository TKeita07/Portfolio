PAPA MOLE

2025

Papa Mole est un jeu d'aventure-plateforme 3D inspiré de Splatoon. Le joueur incarne une taupe capable de creuser pour surmonter les différents dangers et retrouver son fils enlevé par des robots.


Task Overview

  • Conception complète de 2 des 4 niveaux dans le jeu : Level Blockout, Player guiding, Flowchart, etc.
  • Programmation et conception des différents ennemis : Attaques, Enemy Alert System, Enemy Behavior Tree AI, gym metric of the AI's, etc.
  • Conception des mécaniques et du système de combat.
  • Conception de tutoriels.

ELDEN BOAT

2024

Prototype d’une aventure souls-like 3D où le joueur incarne une grenouille embarquée avec son bateau pour explorer un monde humoristique, mais dangereux. Elle se repose sur son attaque faisant aussi office d'esquive afin de surmonter les différents obstacles.


Task Overview

  • Conception complète de 2 des 4 niveaux dans le jeu : Level Blockout, Flowchart, Obstacles placement, etc.
  • Programmation et conception des différents ennemis de base : Attacks, Enemy Alert System, Enemy Behavior Tree AI, etc.

DEBUG CITY

2025

Debug City est un jeu de puzzle coopératif en top down, centré sur le mouvement et la réflexion rapide, créé pour le Gamelab Ubisoft 2025. Afin de détruire les différents virus, les joueurs doivent trouver un moyen de rediriger les projectiles ennemis pour les retourner contre eux afin de les détruire.


Task Overview

  • Conception et implémentation de tous les puzzles (niveaux)
  • Documentation et gestion du projet Méthode Agile, playtest organisation, GDD, Notion, etc.
  • UI/UX design et conception du tutoriel
  • Designed and prototyped the game mechanics: Coop interactions, player skills and the puzzles.
